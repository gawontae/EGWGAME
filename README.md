# 🏝️ Gawon’s Idle Island – Python Idle Game Prototype
**Gawon’s Idle Island**는 Python으로 구현한 간단한 **방치형(Idle) 섬 경영 게임**입니다.  
플레이어는 섬 위에 건물을 지으며 골드, 목재, 돌을 생산하고, 섬을 성장시키게 됩니다.

<details>
<summary>🏝️ Gawon’s Idle Island 게임 설명 펼치기</summary>

### ✅ 1. 전체 개요

- `IdleIslandGUI` 클래스가 전체 GUI + 게임 진행을 관리함  
- `GameState`가 **자원(gold / wood / stone)**과 **건물 보유 수량**을 들고 있음  
- `BuildingType`으로 각 건물의 **가격 / 성장 계수 / 초당 생산량 / 생산 자원 / 색상**을 정의  
- 실제 시간(or 앞으로 감기)을 기준으로 자원이 자동으로 늘어나는 **방치형(Idle) 구조**

### 🌊 2. 자원 & 건물 시스템

#### 🔹 자원 구조 (`GameState.resources`)
- `gold`: 시작 자원 50.0  
- `wood`: 0.0  
- `stone`: 0.0  

게임이 돌아가는 동안, 건물들이 초당 자원을 계속 생산해서 이 값들이 올라감.

#### 🔹 건물 타입 (`BuildingType`)
현재 세 가지 건물 정의:

1. **금광**
   - 설명: 섬 깊은 곳에서 금을 캐는 광산  
   - 기본 가격: 10 gold  
   - 비용 증가 계수: 1.15배  
   - 생산: **gold 1.0 / 초**

2. **제재소**
   - 설명: 나무를 잘라 목재 생산  
   - 기본 가격: 30 gold  
   - 비용 증가 계수: 1.17배  
   - 생산: **wood 0.8 / 초**

3. **채석장**
   - 설명: 절벽에서 돌 채굴  
   - 기본 가격: 50 gold  
   - 비용 증가 계수: 1.20배  
   - 생산: **stone 0.5 / 초**

> 같은 건물을 많이 지을수록 `base_cost * (cost_multiplier ** 보유개수)` 공식으로 **가격이 점점 비싸짐**.

### 🏝 3. 섬 화면 (Canvas)

- 왼쪽 큰 영역이 **섬을 그리는 Canvas**  
- 바탕: 파란색 바다 + 밑에 타원형 모양의 모래 섬  
- 야자수 두 그루를 장식으로 배치  
- 건물을 지으면:
  - 섬 타원 안의 랜덤 위치에 **작은 집 아이콘**이 그림
  - 건물 타입마다 색깔이 달라서 금광/제재소/채석장 구분 가능

섬은 `draw_island_base()`에서 한 번 그리고,  
현재 건물 개수에 따라 `draw_buildings_on_island()`에서 반복해서 집을 뿌려줌.

### 📊 4. 오른쪽 패널 (건물 정보 & 구매)

오른쪽 `건물 & 시간` 패널에서:

- 각 건물에 대해  
  - 이름  
  - 설명 + 초당 생산량  
  - **보유 개수 표시**  
  - **다음 건물 가격 표시**  
  - **[구매] 버튼**으로 바로 건설 가능  

### 🎮 5. 이 게임의 플레이 감각 요약

- 클릭만으로 즐기는 **섬 경영 방치형 게임**  
- 할 일:  
  - 골드로 금광/제재소/채석장 하나씩 사기  
  - 시간이 지날수록 자원이 자동으로 늘어나는 걸 감상  
  - 자원이 쌓이면 더 많은 건물 구매 → 생산량 가속  
</details>

# 🌾 Gawon’s Valley
**Gawon’s Valley**는 Pygame 기반의 스타듀벨리 스타일 미니 농사 게임으로,
작물 성장·시간 변화·상점·퀘스트·스프라이트 애니메이션까지 포함한 구조로 되어 있습니다.

<details>
<summary>🌾 Gawon's Valley 설명 펼치기</summary>

### ✅ 1. 전체 구조
- **FarmGame 클래스**가 게임 전체 로직을 총괄  
- **타일(밭) / 플레이어 / 작물 / NPC / 상점 / HUD**가 각각 객체로 구성됨  
- **시간 흐름 → 작물 성장 → 수확 → 판매 → 레벨업**으로 이어지는 루프 구조

### ✅ 2. 주요 시스템 요약

### 🌱 타일 & 작물 시스템
- 타일 상태 흐름: **빈칸 → 갈기 → 심기 → 성장 → 완전 성장**
- 작물 종류: **무 / 감자 / 호박**
- 각 작물은 **성장일수·판매가격·스프라이트 색상** 보유
- `growth_progress` 값을 이용해 **성장 애니메이션** 구현
- 수확 시 **골드 획득 + 농사 경험치 획득**

### 👤 플레이어 시스템
- 이동: **←↑↓→**
- 작업: **SPACE(갈기/심기/수확)**  
- 잠자기: **N(다음 날)**
- **골드**, **씨앗 인벤토리**, **선택 씨앗(1~3)** 보유
- **농사 레벨·경험치** 시스템 → 레벨 상승 시 성장 속도 증가

### 🕒 시간 & 낮·밤 변화
- `time_minutes` 기반 **실시간 시간 흐름**
- 하늘은 시간에 따라 **부드러운 그라데이션 변화**
- 1440분(1일) 경과 시 **Day 증가**
- 작물은 시간 흐름으로 **자동 성장**

### 🏪 NPC & 상점
- NPC 근처에서 **E키** → 상점 열림
- 씨앗 구매 가능(1~3번 번호 입력)
- 구매 시 **골드 차감 후 인벤토리 증가**
- **ESC/E**로 상점 닫기

### 📜 퀘스트 시스템
- 기본 퀘스트: **“무 5개 수확하기”**
- 작물 수확 시 퀘스트 **진행도 자동 증가**
- 완료 시 **골드 보상** 지급
- HUD 우측에 **퀘스트 정보 표시**

### 🎨 그래픽 요소
- 시간에 따른 **낮/밤 하늘 그라데이션**
- **잔디 / 밭 / 마을(돌길)** 타일 그래픽
- 작물별 **독립 스프라이트(무·감자·호박)**
- HUD 포함 요소: 날짜, 시간, 골드, 씨앗, 레벨, 퀘스트, 조작키 표시
- 상점은 **반투명 오버레이 UI**

### 🎮 3. 조작키 요약
- **←↑↓→** : 이동  
- **SPACE** : 갈기 / 심기 / 수확  
- **N** : 다음 날로 넘어가기  
- **1~3** : 씨앗 선택  
- **E** : NPC 상점 열기  
- **ESC** : 상점 닫기 / 게임 종료  

</details>
