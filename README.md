# 🏝️ Gawon’s Idle Island – Python Idle Game Prototype
**Gawon’s Idle Island**는 Python으로 구현한 간단한 **방치형(Idle) 섬 경영 게임**입니다.  
플레이어는 섬 위에 건물을 지으며 골드, 목재, 돌을 생산하고, 섬을 성장시키게 됩니다.
